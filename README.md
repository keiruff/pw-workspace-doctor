# ProjectWise Workspace Doctor

ProjectWise Workspace Doctor is a lightweight diagnostic and cleanup utility designed for Bentley ProjectWise environments.

Over time, local ProjectWise working directories can grow to tens or hundreds of gigabytes due to cached CAD files, references, DGNLIBs, workspace configurations, and project resources.

This can lead to:

- Slow application startup
- MicroStation lag
- OpenRoads performance issues
- Excessive disk usage
- User confusion about local workspace contents

## Features

- Workspace size auditing
- Bentley file detection
- Largest file identification
- CSV reporting
- Safe cleanup workflow
- Dry-run mode

## Intended Users

- CAD Managers
- Design Technology Teams
- ProjectWise Administrators
- Engineering IT Support Staff
- Transportation Agencies

## Example
